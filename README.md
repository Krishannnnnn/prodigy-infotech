# Titanic Dataset Visualization

This project visualizes the distribution of passengers using the Titanic dataset.

## Tools Used
- Python
- Pandas
- Matplotlib
- Kaggle Notebook

## Visualizations
1. Histogram showing Age Distribution
2. Bar Chart showing Gender Distribution

## Dataset
Titanic Dataset from Kaggle

## Conclusion
Most passengers were between ages 20–40. The dataset also shows more male passengers than female passengers.
